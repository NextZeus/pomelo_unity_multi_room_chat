项目说明
===
使用pomelo作为服务端，unity5.4版本开发的一个多人在线的即时通讯群聊项目，（支持不同的房间，每个房间可以有多人实时群聊）


项目开发环境
===
* nodejs5.8及以上

* pomelo1.2.3

* unity5.4版本
 
* webstorm , vs2015
